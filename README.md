Проект **cooKuzya** — это Telegram-бот для поиска и отображения рецептов. Он помогает пользователям находить блюда по названию, добавлять их в избранное и сохранять понравившиеся рецепты для удобного доступа в будущем. Бот разработан на Python и использует библиотеку для работы с Telegram.

### Основные функции:
- Создание своих рецептов.
- Добавление рецептов в избранное.
- Возможность изменять блюда.
